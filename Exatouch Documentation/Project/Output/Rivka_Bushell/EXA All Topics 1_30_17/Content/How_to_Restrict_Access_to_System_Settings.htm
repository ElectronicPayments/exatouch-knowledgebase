How to Restrict Access to System Settings

By default all users have access to Exatouch System Settings. You can restrict Settings to Manager-Only access by following these steps:

1. From Home, click or tap Settings.

2. Select System.

3. In the Security section, select the Access Controls check box.

4. Once Access controls is checked, you can select or deselect Scramble PIN pad, PIN timeout, and Quick Lock according to your preference. Be sure Access controls remains checked.

4. Click Save.
