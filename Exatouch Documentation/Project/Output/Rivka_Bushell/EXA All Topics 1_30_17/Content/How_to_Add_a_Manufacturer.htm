How to Add a Manufacturer

Adding a manufacturer to your system will allow you to create purchase orders for items and submit the POs to the manufacturer.

Manufacturers can be added to your system from two locations:

From the New Item screen

1. Access the New Item screen from Home > Items > Item Details

2. Click or tap

 

3. Select Add Line.

4. Type in the manufacturer's name.

Add additional names to your manufacturer list by selecting Add Line.

Delete a manufacturer from the list by selecting Delete next to the manufacturer's name.

5. Click Save to save your updated list.

 

From System settings

1. From Home, click or tap Settings.

2. Select System.

3. Click the Custom Labels tab.

4. From the Type drop-down menu, select Manufacturer.

5. Select Add Line.

6. Type in the manufacturer's name.

Add additional names to your manufacturer list by selecting Add Line.

Delete a manufacturer from the list by selecting Delete next to the manufacturer's name.

7. Click Save to save your updated list.
