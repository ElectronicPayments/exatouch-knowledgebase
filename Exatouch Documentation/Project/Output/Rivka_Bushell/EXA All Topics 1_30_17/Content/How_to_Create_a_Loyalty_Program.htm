How to Create a Loyalty Program

A loyalty program can help ensure customer retention and satisfaction. If you wish to create a loyalty program, Contact Technical Support and ask them to activate the Loyalty Program in your Exatouch system.

The Exatouch Loyalty Program is based on points. You'll set up a number of points that customers can earn for every dollar spent.

For example, you can assign 10 points for every dollar. So if your customer spends 5 dollars, they earn 50 points in the loyalty system.

The Loyalty Program also gives customers the opportunity to redeem their points. You set the point redemption value.

For example, you can allow customers to redeem 100 points for 1 dollar. This means that for every 10 dollars spent, they can redeem 100 points.

Follow these instructions to set up a Loyalty Program:
  1. From Home, click or tap Customers.
  2. Select Loyalty.

    3. Note: If you are unable to select the Loyalty option, Contact Technical Support and ask them to activate the Loyalty Module.

  3. Fill in the relevant fields. Be sure to set up point values and choose categories you wish to include in the loyalty program.

    4. Notable Fields:

    Earn: Set up the number of points customers earn for every dollar.

    For example, 10 Points per Dollar.

    Redeem 1 Dollar per: set up the number of points clients need to have to redeem 1 dollar.

    For example, for every 100 points, redeem 1 dollar.

    Expire Time: Determine if a customer's loyalty points expires. From the Expire time drop-down menu, choose between No Expire to 24 months.

    All Customers: Select the All Customers option if you wish to enroll all customers in the loyalty program.

    Member Only: Select this check box if you wish to enroll members only in the loyalty program.

    Points for Joining: In additional to earning points by purchasing your products, customers can earn points by simply joining the program. In the Points for Joining field type in the number of points customers earn by joining the loyalty program.

    Points for Check: In addition to earning points by purchasing your product, customers can earn points by checking in to the system through the Loyalty Kiosk. In the Points for Check field type in the number of points earned when a customer checks into the Loyalty Kiosk.

    Refunds: Gift Cards/Certs only: if this box is checked, customers earn gift cards as their reward for redeeming points. For example, for every $100, customers will earn a $10 gift card.

    All Categories: If this box is checked, customers can earn points by purchasing any and all items in your system. You can restrict the earning of points to only specific items by deselecting this check box.

    Forecolor: This field allows you to define the Loyalty Kiosk background for your tablet. A Loyalty Kiosk is the location used for clients to check into their loyalty account.

    Items Categories and Services Categories: Customers can earn points by purchasing items within categories. Select which categories are designated for the loyalty program in the Items Categories and Services Categories sections.

    For example, you can run a special on the Grocery category one week, and clients can earn points that week by purchasing various grocery items listed within that category.

    Note: To designate weekly product specials which earn loyalty points, consider creating a Loyalty Product category. Then you can populate the Loyalty Product category each week with items you wish to place on special. Each week you can offer different items as specials.

    To create a new Loyalty Product category, see How to Create Categories and Subcategories.

    To add new items to your system, see Add an Item.

  4. Click or tap Save.

Tip! The Loyaty Program and Membership Club Program can be used together.
