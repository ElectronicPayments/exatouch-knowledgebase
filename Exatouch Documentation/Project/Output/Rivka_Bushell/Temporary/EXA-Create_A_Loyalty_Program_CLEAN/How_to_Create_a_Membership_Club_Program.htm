How to Create a Membership Club Program

Membership program gives you the ability to reward frequent purchasers. Membership clubs give customers a percentage discount off the purchase of their entire order. At this time club memberships offer discounts on entire purchases; no exclusions are available at this time.

Membership clubs are created in the Loyalty section. Multiple clubs can be created and maintained at the same time. Customers gain entry to a club by purchasing an item you designate as the Membership Item.

If you wish to create a membership club program Contact Technical Support and ask them to activate the Loyalty Module.

Follow these instructions to create a Membership Club Program:

1. From Home, Customers.

2. Select Loyalty.

Note: If you are unable to select the Loyalty option, Contact Technical Support and ask them to activate the Loyalty Module.

3. Select Add Club.

4. Fill in Club Membership details.

Notable Fields:

Name: Enter the club's name.

Discount Percent: Fill in the percentage discount that customers receive. For example 20.00 will give customers an automatic 20% discount off their entire order.

Term: Set an amount of time that club membership remains active. Choosing No Expire will leave membership active with no expiration date.

Membership Item: Specify the membership item in the Membership Item field. An existing item within your system can be designated as a membership item, or you can create a new membership item for purchase.

Note: There are several ways that you can set up Club Membership:


1. Encourage customers to purchase a specific item by giving them a Club Members hip when they purchase that item.

For example, set up a Star Club for customers who purchase a cell phone. Designate "cell phone" as the Membership Item. When a customer purchases a cell phone, they gain membership to the Star Club. This program encourages clients to purchase cell phones.

2. Clients directly purchase a Club Membership.

For example, set up a Premiere Club which entitles customers to receive 5% off their entire order. To set this up, create a new category called "Membership", and a new item called "Premier Membership". Designate "Premier Membership" as the Membership Item, and assign the item a retail price of $100. When customers purchase "Premier Membership" for $100, they will automatically receive 5% off of their purchase price, deducted automatically at the register.

To create a new Category called "Memberships", see How to Create Categories and Subcategories.
To create a new Item called "Premier Membership", see Add an Item.

5. Save.

Tip: The Loyalty Program and Membership Club Program can be used together.
