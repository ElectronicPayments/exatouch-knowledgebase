Set Tax Rate for an Individual Service

Tax rates can be set when you first create a service, or tax rates can be set for an existing service.

Following are instructions on how to set a tax rate for an existing service:

1. From Home, Items.

2. Select Services Details.

3. Choose the List tab

4. on the desired service on the Services List. If you have many items, you may need to scroll through a number of items to find the desired item. On the touchscreen, move your finger up and down to scroll through the ServicesList.

4. In the Tax section, check one or multiple Tax Groups.

5. Click Save.

Note: Multiple Tax groups can be applied to a single item.
