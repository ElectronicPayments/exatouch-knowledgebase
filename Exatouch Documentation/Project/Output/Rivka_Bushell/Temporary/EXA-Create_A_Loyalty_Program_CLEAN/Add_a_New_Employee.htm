Add a New Employee

When you hire a new employee or add a new employee to the payroll, you may want them to use .

Add an employee to Exatouch by following these steps:

1. From the Home screen, Staff.

2. Select Details.

3. Fill in the employee's information.

4. k Save.

Notable Fields:

Security PIN: This is the 4-digit pin which the staff member will use to login, clock in and clock out.

ID Number: Can refer to a driver's license number or a state issued ID.

User Register Defaults: Fill in Category and SubCategory preferences in this section so that when the employee accesses the register, specific categories and subcategories appear by default.

Enable Calendar: This option is checked by default. When an employee's calendar is enabled, they have access to calendar scheduling .

Software access: This option is checked by default. When this is option is unchecked the employee will not have access to the software; they will only be able to clock in and clock out.
