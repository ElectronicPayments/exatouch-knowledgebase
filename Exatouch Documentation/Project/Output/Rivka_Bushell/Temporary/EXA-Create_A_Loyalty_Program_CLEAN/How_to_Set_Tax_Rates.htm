How to Set Tax Rates

Tax rates are defined in Groups. You can create one or multiple Tax Groups.

For example, taxes applied to soda may be different from taxes applied to beer. So you would create Group A: Tax rates for soda and Group B: Tax rates for beer.

1. From Home, Taxes.

2. Fill in Group A - Tax rates first. Click/tap into the tax fields and type in the desired tax rate.

3. Enter the tax rates for City tax rate, County tax rate, State tax rate, and Federal tax rate where applicable.

Notable Fields:

Select the Default for items check box to apply the Group's tax rules to all items. Different tax rules can be applied later to individual items.

See: Set Tax Rate for an Individual Item

Select the Default for services check box to apply the Group's tax rules to all services. Different tax rules can be applied later to individual services.

See: Set Tax Rate for an Individual Service

4. Create additional Tax Groups by filling in tax rates for each group (i.e. Group B, Group C...)

5. Click Save

Note: Just as you can apply multiple tax rates to a single item, you can apply multiple Tax Groups as your System Default. For example, you can check Default for items in both Group A and Group B.
