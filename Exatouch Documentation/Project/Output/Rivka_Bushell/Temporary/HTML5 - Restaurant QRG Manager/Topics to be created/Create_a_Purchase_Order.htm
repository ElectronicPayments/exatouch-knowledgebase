Create a Purchase Order

 

Purchase Orders (PO) are a request for purchase from a vendor.

1. From Home, Purchasing.

2. Select PO Details.

3. Select a vendor you wish to create a PO.

4. Manually add the items you wish to add to the PO. Any reorder quantity or cost applied to the item will be added to the Purchase Order list.

5. Select Expected Date for when the order will be received.
