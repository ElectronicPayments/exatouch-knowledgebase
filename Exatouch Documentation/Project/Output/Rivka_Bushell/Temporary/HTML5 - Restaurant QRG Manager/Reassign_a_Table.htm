Reassign a Table

Sometimes you need to reassign a table to another wait staff member. If your restaurant uses the Open Flood Plan, you can reassign tables. Tables are reassigned from the Floor Plan:

1. From Home, Register.

2. Select the Floor Plan tab.

3. From the Floor Plan, on the table you want to reassign

4. Select Reassignfrom the pop-up menu.

4. From the pop-up box, click on the wait staffer's name that you want to reassign the table to. If there are many wait staffers listed, you can search for a name by typing into the Start typing search string field.

5. Check to see that the table was reassigned by clicking again on the table, and by reviewing the table information in the pop-up box.
