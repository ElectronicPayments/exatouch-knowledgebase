Assign a Vendor to an Item

Assigning a vendor to an item gives you the ability to create a Purchase Order (PO) for that item.

1. From Home, Items.

2. Select Items List .

3. Choose an item by tapping on the item name. You may have to scroll through the Items List with your finger to locate your selection.

4. Click the Inventory tab.

4. Add.

5. Tap vendor's name to select the vendor.

Tip: If the vendor's record does not yet exist in your system, select New Vendors to quickly add a vendor.

Also see: Add a Vendor

6. Select Save.

Note: In , items are always linked to vendors. If no vendor is assigned to an item, a default vendor wil be applied. You can set a default vendor by clicking Set Default in your Vendor(s) List. If only one vendor exists in your system, that vendor becomes the default vendor; if no vendor has been added to your system, a default vendor called No Vendor is created.
