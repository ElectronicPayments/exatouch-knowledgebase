Split a Check by Item

Customers may want to split their order into multiple checks:

1. From the Register, Split by item.

2. Click to highlight the desired item.

3. the order box (i.e. Order 2) that youd like to move the item to. Tapping the desired order box will move the item to a separate check.

4. Select Done. This will create new checks for each order.
