Modify Sell Options

Sell options can be set globally or individually. When you set a global sell option, items in your system will automatically default to that sell option. You can apply different sell options to individual items as needed.

Set Global Sell Option:

1. From Home, Settings.

2. Select Preferences.

3. In the Inventory section, select click the Sell options drop-down menu.

4. Select from Stop selling at 0, Go negative, or Not an inventory item.

5. Click Save to apply your sell option globally.

Set Sell Option for a Single Item:

1. From Home, Items.

2. Select Items List.

3. Click on the item you want to modify. To locate an item on a longer list, scroll with your finger on the touchscreen until you find the item.

4. Click on the Details Tab to view Items Detail.

5. In Sell options field, click the drop-down menu.

6. Select from Stop selling at 0, Go negative, or Not an inventory item.

7. Select the Save tab.

Note: Select the Alert check box to receive a system alert which will notify you when a customer orders an item which is no longer in stock.

Notable Options:

Stop selling at 0 will limit the sale of the item once inventory reaches 0.

Go negative will allow the sale of the item if the inventory is 0. This option is useful for those times when your stock exceeds your inventory total. You can always modify your inventory to reflect additional stock.

See: Modify Inventory Level

Not an inventory item will allow the sale of an item which is not tracked as an inventory item. For example, drop-ship items which are not kept in stock and are special ordered, are not an inventory item. Most restaurants do not consider their menu selections inventory items.
