Create a Purchase Order

Purchase Orders (PO) are a request for the purchase of items from a vendor.

Tip! Before an item can be ordered, that item needs to to be linked to the vendor who supplies it. If your item has not yet been linked to a vendor, follow these instructions to Assign a Vendor to an Item

Follow these steps to create Purchase Order:

  1. From Home, Purchasing.
  2. Select PO Details.
  3. On the Create PO Screen, select a vendor either by typing the vendor name into the Vendor field, or by clicking to search and select a vendor from the Select Vendor popup
  4. Select a vendor you wish to submit the PO to. Note that items associated with the vendor will be displayed in the Vendor's Items list.
  5. Manually add the items you wish to add to the PO. Any reorder quantity or cost applied to the item will be added to the Purchase Order list.
  6. Select Expected Date for when the order will be received.
