How to Create Categories and Subcategories

Category and SubCategory fields help you organize and keep track of your items.

Categories and subcategories are created on the Categories screen.

 

Create a Category:

1. From Home, click or tap Items.

2. Select Categories.

Items Categories is the list of all current categories. If you have not yet entered any Categories, this list may be empty.

3. Add a category by clicking or tapping the first empty line on the Items Categories List.

Tip: Open a touchpad keyboard by double clicking in the empty line, or Click

4. Type the name of a category, and then press Enter.

5. Repeat steps to create additional categories.

 

Create a SubCategory:

SubCategories are secondary options within a category.

1. Select a the desired category by clicking to highlight the category name. Subcategories will be created for this designated category.

 

2. In the SubCategories column, click or tap the first empty line on the SubCategories list.

3. Type the name of a subcategory, and then press Enter.

4. Repeat steps to create additional subcategories.
