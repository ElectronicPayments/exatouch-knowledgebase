Split a Check by Item

Customers may want to split their order into multiple checks:

  1. From the Register, click or tapSplit by item.
  2. Click to highlight the desired item.
  3. Click or tap the order box (i.e. Order 2) that you'd like to move the item to. Tapping the desired order box will move the item to a separate check.
  4. Repeat tapping to move additional items you'd like to move to the separate order.
  5. Select Done. This will create new checks for each order.
