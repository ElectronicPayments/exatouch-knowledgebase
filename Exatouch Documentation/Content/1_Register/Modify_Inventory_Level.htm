Modify Inventory Level

There are several ways to modify your inventory level.

Adjust inventory on Items Detail Screen

The most basic way to modify inventory level is through the Items Detail screen:

1. From Home, Items.

2. Select Items List.

3. Click the item you'd like to modify.

4. Select the Inventory tab.

5. Adjust Inventory Quantities to reflect available stock.

Note: If an item is designated as a "Non-Inventory Item" you will be unable to adjust the inventory amount for the item. To change an item's from "Non-Inventory" status, see: Modify Sell Options
