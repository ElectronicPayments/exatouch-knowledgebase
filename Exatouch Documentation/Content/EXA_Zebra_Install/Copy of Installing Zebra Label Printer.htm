Installing Zebra Label Printer for Exatouch
After plugging in USB cable to Exatouch POS USB port, log into Reserved/Windows

On Internet Explorer browser type in:
http://www.zebra.com/setup where you will locate the Zebra Setup download link.

 

 

Select the “Download” link.

After selecting the download you will be prompted to Register for an account or Sign in.
Register your account and sign in.

 

 

 

 

 

After signing in you will be asked to Accept and Begin Download for Zebra Setup software, select.
When asked to Run or Save: Run the application for download.


 

 

 

Allow Install Wizard for Zebra Setup.

 

Once Install Wizard Application is downloaded, select “Next” through the prompts.



Accept the Terms of the License Agreement and Select Next.

Folder for Zebra Setup will be created, Select Next.



Allow install application for “All Users”, Select Next.

 

The final page of the Install Wizard will appear to install Zebra Setup Utilities.
If changes need to be made select “Back”.

To continue with configuration/install please select Next.


Allow Install Wizard to configure Zebra Setup Utilities.

Please check off that you are using the latest version (as downloaded off the Zebra Setup Link)



Install Wizard has completed.
Please check off the Run Zebra Setup Utilities and “Finish”.


System Prepare Wizard may ask which port the Zebra Printer is connected to, to continue Installing Printer Driver.

Select USB port and Next.


Confirm Printer Driver Install automatically by selecting “Finish”.

Zebra Setup Utilities will appear (which will also be uploaded to the Windows Desktop).
On Zebra Setup Utilities we will need to Install the correct printer model that will be used for Exatouch Label Printing.


The correct Printer will be ZDesigner ZD410-203dpi ZPL


If not listed below in the printer’s window, select Install New Printer.

 

 

 

 

 

 

 

Once Install New Printer is selected, Printer Install Wizard will ask you to begin the installation process.
Select Next to continue.



On Installation Options select Install Printer.

On the Select Printer window you will see a list of Zebra printer models.
Select “ZDesigner ZD410-203dpi ZPL” and Next.

 

 

 

 

 

Selecting Next through the prompts will allow this Printer type to be installed on to your Zebra Setup Utilities window.

Listing the Manufacturer, Model and Location (port).

 

 

 

 

 

To Confirm the printer and drivers were successfully installed it will also be listed in the Windows “Devices and Printers” which can be navigated through Windows Control Panel.

 

 

 

 

 

 

 

 

 

 

 


Once confirmed that the Zebra/ZDesigner Printer is now installed, log into Exatouch to set the printer settings. On Exatouch homescreen select the Settings Module.

.

In the Settings module, select Print module.

In the Print module you will define the Printer that the Labels will be assigned to.

 

Select ZDesigner ZD410-203dpi ZPL printer to be assigned.
