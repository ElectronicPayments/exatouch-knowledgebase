How to Set a Default Vendor for an Item

In , items are always linked to vendors.

To set a default vendor for an item from the Item's Detail page:

1. From Home, Items.

2. Select Items List.

3. Select the name of the item. If there are many items to choose from, you may have to scroll through the Items List, or you can search for the name in the Start typing search field.

4. Select the Inventory tab.

5. In the Vendor(s) section, choose Add.

6. Choose a vendor by clicking to highlight. If your vendor is not listed, click New Vendors to Add a Vendor.

7. The vendor should now appear in the Vendor(s) section. Click to highlight the vendor.

8. Set Default.

Add a Vendor from Item Details:

You can add a vendor as you are editing or adding an item in the Item's Details

1. Navigate to Item Details. Home > Items > Item Details.

2. Choose the Inventory tab.

3. From the Vendor(s) section, click Add.

4. Select New Vendors.

Note: Set an item's default vendor You can set a default vendor by clicking Set Default in your Vendor(s) List. If no vendor is assigned to an item, a default vendor will be applied.. If only one vendor exists in your system, that vendor becomes the default vendor; if no vendor has been added to your system, a default vendor called No Vendor is created.
